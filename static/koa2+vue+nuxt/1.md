

- [nvm](https://github.com/creationix/nvm)

使用nvm管理node。

```
// 安装nvm
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
nvm install v8.1.2
nvm use v8.1.2
node -v

nvm alias default v8.1.2  // node命令的默认版本是8.1.2 
```
