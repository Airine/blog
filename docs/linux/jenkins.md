# jenkins 文档笔记

-   [官网文档](https://jenkins.io/zh/doc/)

## 概念

Jenkins 是一款开源 CI&CD 软件，用于自动化各种任务，包括构建、测试和部署软件。

Jenkins 支持各种运行方式，可通过系统包、Docker 或者通过一个独立的 Java 程序。

## 安装

### 前提

-   10G 硬盘空间，512 MB 内存
-   java8
-   Docker

### 步骤

1. 下载 jenkins，进行下载目录
2. 运行 `java -jar jenkins.war --httpPort=8080`
3. 打开浏览器`http://localhost:8080`
4. 安装
